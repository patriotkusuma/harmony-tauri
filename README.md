# Harmony Laundry Point of Sale (Harmony-Tauri)

**Harmony Laundry POS** adalah aplikasi desktop modern yang dirancang khusus untuk mengoptimalkan operasional bisnis laundry. Aplikasi ini menggabungkan antarmuka web yang responsif menggunakan **React** dengan ketangguhan aplikasi desktop melalui **Tauri** dan **Electron**.

### Fitur Utama:
- 📊 **Dashboard & Metrik Real-time**: Pantau performa outlet secara instan.
- 📦 **Manajemen Inventaris**: Pelacakan stok otomatis dengan peringatan stok menipis (*Low Stock Alerts*).
- 🏷️ **Integrasi RFID**: Audit dan pelacakan pakaian menggunakan teknologi RFID untuk menghindari kehilangan barang.
- 📑 **Laporan Operasional**: Analisis kinerja karyawan, status pesanan, dan produktivitas harian.
- 💬 **Notifikasi WhatsApp**: Pengiriman pesan otomatis ke pelanggan terkait status cucian.
- 🔐 **Manajemen Karyawan & Akses**: Kontrol shift dan logging aktivitas kasir.

### Tech Stack:
- **Frontend**: React.js, Reactstrap, Chart.js
- **Desktop Wrapper**: Tauri & Electron
- **State Management**: Zustand & React Query
- **Data Fetching**: Axios
